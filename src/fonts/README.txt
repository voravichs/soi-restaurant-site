Pathway Extreme Variable Font
=============================

This download contains Pathway Extreme as both variable fonts and static fonts.

Pathway Extreme is a variable font with these axes:
  opsz
  wdth
  wght

This means all the styles are contained in these files:
  PathwayExtreme-VariableFont_opsz,wdth,wght.ttf
  PathwayExtreme-Italic-VariableFont_opsz,wdth,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
